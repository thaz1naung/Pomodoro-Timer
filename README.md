# 🍅 Pomodoro Timer with Vibes

A focused, calm Pomodoro timer built with vanilla HTML, CSS, and JS. No frameworks, no build step, no backend.

## Features

- **25/5/15 timing** — Focus, Short Break, Long Break (after every 4 pomodoros)
- **3 mood themes** — Focus (red-orange), Chill (soft blue), Deep Work (purple)
- **Streak counter** — Track today's completed pomodoros, persisted in localStorage
- **Web Audio chime** — Three-tone bell on timer end, with mute + volume control
- **Animated background** — Subtle floating blob ambient backdrop per theme
- **Responsive** — Works on everything from iPhone SE to desktop, landscape-aware

## How to run

Open `index.html` in any browser. That's it.

## Deploy

```bash
vercel --prod
```

## Tech

HTML + CSS + JS. Zero dependencies. Zero build step. Google Fonts CDN for Inter + JetBrains Mono.
